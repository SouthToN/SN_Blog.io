Day07： 二进制中1的个数 & 数值的整数次方

2020-02-20

一、二进制中1的个数

请实现一个函数，输入一个整数，输出该数二进制表示中 1 的个数。例如，把 9 表示成二进制是 1001，有 2 位是 1。因此，如果输入 9，则该函数输出 2。

示例 1：
输入：00000000000000000000000000001011
输出：3
解释：输入的二进制串 00000000000000000000000000001011 中，共有三位为 '1'。

示例 2：
输入：00000000000000000000000010000000
输出：1
解释：输入的二进制串 00000000000000000000000010000000 中，共有一位为 '1'。

示例 3：
输入：11111111111111111111111111111101
输出：31
解释：输入的二进制串 11111111111111111111111111111101 中，共有 31 位为 '1'。

来源：力扣：二进制中1的个数

思路1：直接统计手动十进制转二进制过程中1的个数（but: LeetCode提交未通过🙃）
code1:

public class Solution {
    public int hammingWeight(int n) {
        int count=0;
        while(n!=0){
            if(n % 2 == 1){
                count++;
                n /= 2;
            }
        }
        return count;
        
    }
}

思路二：位运算
code2：

public class Solution {
    // you need to treat n as an unsigned value
    public int hammingWeight(int n) {
        if(n==0) return 0;
        return hammingWeight(n&(n-1))+1; 
    }
}

二、数值的整数次方

实现函数double Power(double base, int exponent)，求base的exponent次方。不得使用库函数，同时不需要考虑大数问题。

示例 1:
输入: 2.00000, 10
输出: 1024.00000

示例 2:
输入: 2.10000, 3
输出: 9.26100

示例 3:
输入: 2.00000, -2
输出: 0.25000
解释: 2-2 = 1/22 = 1/4 = 0.25

来源：力扣：数值的整数次方

思路1：直接循环（but: x=0.00001 n=2147483647过不了🙃）
code1:

class Solution {
    public double myPow(double x, int n) {
        if(n==-1) return 1/x;
        if(n==0) return 1;
        double result=1;
        if(n<0){
            for(int i=1;i<=-n;i++){
                result*=x;
            }
            return 1/result;
        }else{
            for(int i=1;i<=n;i++){
                result*=x;
            }
            return result;
        }
    }
}

code2：

class Solution {
    public double myPow(double x, int n) {
        if(n==-1) return 1/x;
        if(n==0||x==1) return 1;
        double num=1;
        long N=n;
        if(N<0){
            N=-N;
            x=1/x;
        }
        while(N>0){
            if(N%2==1) num=num*x;
            N=N>>1;
            x=x*x;
        }
        return num;
    }
}
