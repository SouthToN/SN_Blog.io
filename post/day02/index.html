一、替换空格

请实现一个函数，把字符串 s 中的每个空格替换成"%20"。
示例 1：
输入：s = "We are happy."
输出："We%20are%20happy."

来源：力扣：替换空格

code1:

class Solution {
    public String replaceSpace(String s) {
        return s.replace(" ", "20%");
    }
}
code2:
class Solution {
    public String replaceSpace(String s) {
        StringBuffer sBuffer = new StringBuffer();
		char[] cs = s.toCharArray();
		for (int i = 0; i < cs.length; i++) {
			if (cs[i]!=' ') {
				sBuffer.append(cs[i]);
			} else {
				sBuffer.append("%20");
			}
		}
		return sBuffer.toString();
    }
}

二、从尾到头打印链表

输入一个链表的头节点，从尾到头反过来返回每个节点的值（用数组返回）。

示例 1：
输入：head = [1,3,2]
输出：[2,3,1]

来源：力扣：从尾到头打印链表

code1:

/**
 * Definition for singly-linked list.
 * public class ListNode {
 *     int val;
 *     ListNode next;
 *     ListNode(int x) { val = x; }
 * }
 */
class Solution {
    public int[] reversePrint(ListNode head) {
       	ListNode cur = head;
		int len = 0;
		while (cur != null) {
			len++;
			cur = cur.next;
		}
		int[] rp = new int[len];
		while (head != null && len > 0) {
			rp[len - 1] = head.val;
			head = head.next;
			len--;
		}
		return rp; 
    }
}
